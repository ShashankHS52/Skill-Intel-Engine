# **App Name**: Skill Intel Engine

## Core Features:

- KPI Ticker: Display real-time key metrics, like total registered users, critical skill gaps and aligned training programs, for immediate insights into workforce trends.
- Geographical Heatmap: Visualize unemployment rates and skill shortages across regions, with filtering by district/block for targeted policy intervention.
- Automation Risk Alert: Identify sectors/skills at highest automation risk and the affected population percentage to proactively address workforce transition.
- Skill Gap Analyzer: Analyze skill gaps by sector, region, and proficiency, suggesting policy recommendations via AI, like funding new training centers or employer subsidies; the LLM will use its tool to incorporate the regional, proficiency, and other selected user defined considerations to recommend the course of action.
- Training Program Management: Map training program curricula to a Standardized Skill Taxonomy and calculate an Alignment Score to ensure effective skill gap resolution in target regions.
- Data Quality Monitoring: Monitor data quality, flagging duplicate profiles, incomplete registrations, and skills with low confidence scores for manual review, maintaining a clean, reliable dataset.
- Scenario Testing: Simulate the impact of economic changes (e.g., new factories) on projected skill demand to enable proactive workforce planning and policy adjustments; the LLM will use its tool to incorporate factory location, labor types required, etc. to predict how policy makers should react.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey professionalism and trustworthiness.
- Background color: Light teal (#E0F8F8), a muted version of the primary color, to ensure readability and a calm visual environment.
- Accent color: Coral (#FF7F50) to draw attention to key actions and alerts.
- Body and headline font: 'Inter', a sans-serif font, to maintain a modern, machined, objective, neutral look for ease of reading and data presentation.
- Use clean, consistent icons to represent different data categories and actions, enhancing usability and navigation.
- A clear, grid-based layout with distinct sections for KPIs, analytics, and management tools to facilitate easy data access and decision-making.
- Subtle transitions and animations for data updates and user interactions to provide feedback without being distracting.
